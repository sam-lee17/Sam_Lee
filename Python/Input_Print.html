September 13, 2020
Python

Input function

The input function allows the user to type in a new value every time one runs the code.

To use this function, you need a variable.

x = input()

You can write the description inside the parentheses using quotation marks (" ").

x = input("Please enter a number")
Please enter a number

Something that you should keep in mind is that the value of the input is always in string type.

So if you want to change the entered value to a number, you can use an int( ) function.

x = input(20)
x = int(x)

The '20' is a string type at first but it changes to a number type by using int( ).

To verify that it is a number, or to check its variable type, you use the type( ) function.

x = int(input())
print(type(x))

20
class 'int'

The output indicates that '20' has been converted to a number.

Q1. At a cafe you own, espresso and americano cost $2.5, and latte costs $3. If you have sold 10 cups of espresso, 5 cups of americano, and 15 cups of latte, how much did you earn today? (Keep on mind that the number of cups sold varies every day)

A1.

Let's define the number of espressos sold as x, the number of americanos sold as y, and the number of lattes sold as z.

Since the number of cups sold varies every day, use the input function to we could enter a new value every time when calculating the earnings. Here, we need to be aware that the value is always in string type; we want the value to be in a number type. This can be done using the int function.

x = int(input())
y = int(input())
z = int(input())

 

To calculate the total gain, we need to multiply the number of cups sold by the cost of a cup for all three drinks and add the three results.

espresso = x * 2.5
americano = y * 2.5
latte = z * 3
print("Today's total earning is: $", espresso + americano + latte)

Q2-1. Create a program that converts Fahrenheit to Celsius.

      Formula: C = 5/9 (F - 32)

A2-1.

The program should be able to convert any value inputted in Fahrenheit to Celsius.

F = int(input())

Here, the computer receives a value and converts it into a number type.

Next, we use the formula given above to calculate the value in Celsius.

C = 5/9* (F - 32)
print("Current weather in Celsius is: ", C)

Loop Statement 2 (While Loop)

August 14, 2018
Python

While loops

while (condition): 
    (statements that you input)

The basic format of while loops follows those of other conditional and loop statements: indicating with a colon and an indent. The most prominent characteristic of the while loop is that it continues to execute the statements when it satisfies the conditions. As a result, some conditions might make your computer fall into the loophole of the neverending repetition of the statement.

while True:
    print(1)

This code will print out one forever. Therefore, it is essential to be conscious of what you are entering as your condition.

count = 10
while count > 0:
    print(count)
    count = count - 1

This condition allows the computer to exit the code after printing out from 9 to 0 consecutively because of how the condition is set. The count variable is set as 10 in the beginning, and the count is reduced by one every time you print out.

count = 10
while count > 0:
    count = count - 1
    if count == 5:
        break
    print (count)
9
8
7
6

We could also use the break statement to terminate the loop. This code "breaks" the loop when the count variable equals 5, which means it would print out from 10 to 6 instead.

count = 10
while count > 0:
    count = count - 1
    if count == 2:
        continue
    print (count)

The continue statement stops the current process and moves on to the next step. When the count reaches 5, it would not print anything but subtract one from the count, which means that the computer would print out from 9 to 0 consecutively except for 5.

Q. Create a code that prints out a multiplication table of the input using the while loop. (the same question that we saw in the previous post but with while loop instead of for loop)

MT = int(input("which multiplication table do you want to print?"))

First, you do the same thing where you receive input from the user.

You could maintain the same format of multiplying the input MT by 1 to 9 each time, but use the while loop this time.

We could set up the condition using the count variable again so that it would exit the code after multiplying from 1 to 9.

MT = int(input("which multiplication table do you want to print?"))

count = 1
while count < 10:
    print(MT, 'X', count, '=', MT*count)
    count = count + 1

 

If MT is 4, it would print out the below:

4 X 1 = 4
4 X 2 = 8
4 X 3 = 12
4 X 4 = 16
4 X 5 = 20
4 X 6 = 24
4 X 7 = 28
4 X 8 = 32
4 X 9 = 36

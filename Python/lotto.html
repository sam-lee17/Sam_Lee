Lotto generator

August 14, 2018
Python

Q. Make a lotto generator that generates random 6 numbers in the range between 0 and 46 without repetition. (Hint: use for and while loops)

How can we generate random numbers? We can use a predefined function called random.

import random

To use this function, you need to import it at the beginning of the code to indicate that you are bringing information on random functions to the local computer. There are multiple functions encompassed in the random function, but here we are only going to use random.random (), which returns a random decimal number between 0 and 1. Therefore, we are going to multiply by 100 to get a number between 0 and 100.

import random
                                lotto = []

The lotto list would harbor the 6 lotto numbers.

import random
                                lotto = []

                                for i in range(6):
                                    while True:
                                        num = int(random.random ()*100)

Now that we know how to return a random number, we need to repeat the random number returning code 6 times to get 6 different numbers.

import random
                                lotto = []

                                for i in range(6):
                                    while True:
                                        num = int(random.random ()*100)
                                        if num > 0 and num < 46:
                                            if num not in lotto:
                                                lotto.append(num)
                                                lotto.sort()
                                                break

This code will append the random number generated only if it is between 0 and 46 (the range of lotto numbers stated in the question) and not in the lotto list (since we need to avoid repetition). Then, the computer presents the lotto list in consecutive increase form.

import random
                                lotto = []

                                for i in range(6):
                                    while True:
                                        num = int(random.random ()*100)
                                        if num > 0 and num < 46:
                                            if num not in lotto:
                                                lotto.append(num)
                                                lotto.sort()
                                                break
                                print(lotto)

                                [4, 5, 12, 14, 32, 42]

Once we run the code, it would print out 6 random numbers in a list as shown above.

Q. Find out how many times you need to try to win the lotto. Let's say that the winning lotto numbers are 4, 5, 12, 14, 32, 42 from the previous question.

Here, it is similar to the lotto generator, but you need to consider one more element: the winning lotto numbers. As you continue to produce random lotto lists, you need to constantly compare them with the winning lotto list. Also, you need two counting variables: one to count how many tries you had and another to count how many of the randomly generated numbers are equal to the winning lotto numbers.

import random
                                jackpot = [4, 5, 12, 14, 32, 42]
                                num = 0

Similarly, we start by importing the random function. The jackpot is the winning lotto list that you are trying to get and the num is a counting variable indicating the number of attempts.

import random
                                jackpot = [4, 5, 12, 14, 32, 42]
                                num = 0
                                while True:
                                    num = num + 1
                                    lotto = []
                                    count = 0
                                    for i in range (6):
                                        num = int(random.random ()*100)
                                        if num > 0 and num < 46:
                                            if num not in lotto:
                                                lotto.append(num)
                                                break

The count variable counts how many of the randomly generated numbers are equal to the winning lotto numbers. Until here, there is not much difference from the previous question except for some additional variables. 

import random
                                jackpot = [4, 5, 12, 14, 32, 42]
                                num = 0
                                while True:
                                    num = num + 1
                                    lotto = []
                                    count = 0
                                    for i in range (6):
                                        num = int(random.random ()*100)
                                        if num > 0 and num < 46:
                                            if num not in lotto:
                                                lotto.append(num)
                                                break
                                    for i in range (6):
                                        if jackpot[i] in lotto:
                                            count = count + 1
                                    if count == 6:
                                            break

This is the key part of the entire code. You keep comparing each number from the random lotto list and each from the winning lotto list. Then, when all 6 numbers are identical, we exit the loop since we have finally won the lotto.

import random
                                jackpot = [4, 5, 12, 14, 32, 42]
                                num = 0
                                while True:
                                    num = num + 1
                                    lotto = []
                                    count = 0
                                    for i in range (6):
                                        num = int(random.random ()*100)
                                        if num > 0 and num < 46:
                                            if num not in lotto:
                                                lotto.append(num)
                                                break
                                    for i in range (6):
                                        if jackpot[i] in lotto:
                                            count = count + 1
                                    if count == 6:
                                            break
                                print ("How many attempts did it take?:" ,lotto, "times")\

                                How many attempts did it take?: 13998130 times

Lesson learned: don't buy the lotto. I would rather work hard and earn money that is equivalent to the amount of work I have done than to do nothing while dreaming to suddenly become a billionaire by winning the lotto.

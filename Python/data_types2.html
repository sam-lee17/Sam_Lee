List / Tuple / Dictionary / Set

October 18, 2020
Python

List

1. Multiple data are grouped together as a list. (Data type can be different)

2. Square brackets are used to represent a list

3. It has an order, which means indexing and slicing functions are possible.

4. The value of the data can be changed.

5. Data are separated by commas.

 

Indexing

ex)

l = [1, 1.4, 'data', [1,2,3]]
print (1[2])
data

 

Slicing

ex) 

print (l[0:3])
1, 1.4, 'data'

ex)

l = [1, 2, 3, 4],[5, 6, 7]]
print ([0][2])
3

 

Tuple

A tuple is the basic data type used to represent more than one value.

It has the same characteristics as a list except for 2 things, which means indexing and slicing functions can be used.

1. It is represented by a parenthesis instead of square brackets.

2. The values of data cannot be manipulated.

 

If a tuple offers less efficient functions, why do we use it?

Tuple has more memory space and fewer unnecessary functions. So when the number of data increases infinitely, a computer performs much faster with a tuple than a list.

 

Characteristics

Tuple can be changed into a list and vise versa.

ex) 

T = tuple(4, 67.8, 'hi there', [4,246,84])
T = list(T)
[4, 67.8, 'hi there', [4,246,84]]

 

Dictionary

1. Dictionary is represented by braces.

2. There is no order.

3. It consists of keys and values. You can think that you are assigning certain values or a key.

(More than one values can be assigned to one key)

4. The keys are immutable, which means that only string, number, and tuple are possible data types for keys. On the other hand, all data type is available for values.

 

ex) 

dic = {'d':123, 'i':'hello', 'c':[4,5,6]}

ex) How to print out just the items

print (dic['d'])
123

ex) How to append a new set of key and item

dic[7] = ('n','e','w')
print (dic)
{'d':123, 'i':'hello', 'c':[4,5,6], 7:('n','e','w')}

ex) How to change the value of an existing key

dic['i'] = 'hi'
print (dic)
{'d':123, 'i':'hi', 'c':[4,5,6], 7:('n','e','w')}

ex) How to delete a key and its corresponding value

del dic['c']
print (dic)
{'d':123, 'i':'hi', 7:('n','e','w')}

 

Set

1. It is represented by parentheses ( ).

2. It does not have an order nor repetition. So it is commonly used to eliminate any repetition in the set of data.

 

ex)

a = set('hello')
print (a)
'e', 'h', 'o', 'l'

Because the set does not have an order, if you run the previous code once more, you can receive another value such as these indicated below:

'h', 'l', 'o', 'e'

ex)

s = (1,2,3,4,5,6,7,3,5,2,6,8,1,9,4,2,67)
s = set(s)
print (1,2,3,4,5,6,7,8,9,67)

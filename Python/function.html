June 9, 2020
Python

Function

def (parameter):
    (statement)

Above is the basic format of functions. 

We use functions to eliminate repetition in the code by defining the repeating code in a function. For example, if we set the lotto generator in a lotto function, 

import random
def lotto():
    lotto_list = []
    for i in range (6):
        while True:
            num = int(random.random()*100)
            if num > 0 and num < 46:
                if num not in lotto_list:
                    lotto_list.append
                    break
print(lotto_list)

this entire code will be run once you run the lotto () function.

However, the code is not executed unless you call the function again. In other words,

def func():
    a = 1
    print(a)

you create a function 

func ()

1

that prints out 1.

But, 

print (a)

Error: a is not defined

when you try to print (a), you get an error, although you have set a = 1 in the function called func ().

This is because a = 1 is inside the function, a local scope, and the computer does not "remember" the variable anymore as the function ends.

If a is defined as 1 outside of the function, it would be considered as a global scope. This means that the computer "remembers" the variable until the python ends. As a result, the computer would print out 1 without having to call the function.

If you would like the value of the variable a to be identical inside and outside of the function,

def func ():
    global a
    a = 1
    print(a)

you can claim that the variable a inside the function is global.

Then,

func ()
print(a)

2
2

the computer would print out 2 for both scenarios.

 

Return

A return statement may seem identical to the print function since they do the same work in terms of returning a value. However, there is a major difference. 

The print function is a function, which means that it would only return a value when you "tell" the computer to do so.

On the other hand, the return statement defines the value that you get from the function as the function itself.

def lotto ():
    create_lotto = [] # the six random numbers put in a list
    # rest of the lotto code
    return create_lotto

Here, the computer would return the value of create_lotto as lotto (). In other words, the data type of lotto () becomes a list, and now you are able to utilize different functions specific to the list.

 

Parameter / Argument

Parameter / Argument is a variable that is passed into a function and prints out the value of the code depending on the altering value of the parameter. It seems similar to the input function, but a parameter is part of the function. You need to identify the parameter if you included one every time to call the function. In addition, you need to be aware of how many parameters you are including because the code works only if the correct number of parameters are stated when calling the function.

def mul_table (num):
    count = 0
    while True:
        count = count + 1
        print(num*count)
        if count == 9:
            break

This is an example of a code that prints out a multiplication table of the parameter. In this case, the parameter is set as num.

If you call the function with the parameter being 9,

mul_table (9)

the computer would print out the multiplication of 9.

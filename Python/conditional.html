(Control Statement 1)

December 13, 2020
Python

Control Statement

Control Statement allows you to control the order of the programming. In general, the code is executed sequentially when the program is running. Once the control statement is used, however, the sequence of the code is manipulated, and the computer returns the value depending on the influence of the control statement.

Here, there are two types of control statements: a conditional statement and a loop statement. This time, I will explain the control statement.

Conditional Statement

1. Conditional Statements only flow the code when it satisfies the condition.

2. They are represented by a colon and an indent to indicate the portion of the code that belongs to conditional statements.

3. There are three ways to represent the conditions - if statement, elif statement, and else statement.

 

If Statement

If Statement indicates the start of the conditional statement.

a = 1
if a > 0:
    print(1)
if a < 0:
    print(-1)

1

=> If 'a' is greater than 0, print 1. If 'a' is less than 0, print -1. Because 'a' is greater than 0, it satisfies the first condition, and, as a result, the computer prints 1.

 

Elif Statement

Elif Statement has the same function as if statement, but it is used to add more conditions.

a = 4
if a == 1:
    print (1)
elif a == 2:
    print (2)
elif a == 3:
    print(3)
elif a == 4:
    print(4)

4

This single conditional statement encompasses 4 conditions.

 

Else Statement

If the input does not satisfy any conditions mentioned above, it is affected by Else Statement.

a = 5
if a == 1:
    print (1)
elif a == 2:
    print (2)
elif a == 3:
    print (3)
else:
    print(4)

4

As 'a' does not satisfy any of the conditions in if or elif statements, it is automatically considered to satisfy the else statement.

Q1.

A) Create a letter grade determinator based on this grading scale. (A numeric grade is a whole number)

Grading Scale
Letter Grade Numeric Grade (%)
A 90 - 100 %
B 80 - 89 %
C 70 - 79 %
D 60 - 69 %
F 0 - 59 %

 

score = int(input("Please enter your numeric grade"))
if 90 <= score <= 100:
    print("Your letter grade is A!")
elif 80 <= score < 90:
    print("Your letter grade is B!")
elif 70 <= score < 80:
    print("Your letter grade is C!")
elif 60 <= score < 70:
    print("Your letter grade is D!")
else:
    print("Your letter grade is F!")

B) Add another condition that if a student has an A and received an effort mark, he or she will earn an A+.

score = int(input("Please enter your numeric grade"))
EF = int(input("Please enter 1 if you have received an effort mark. If not, please enter 0."))
if 90 <= score <= 100:
    if EF = 1:
        print("Your letter grade is A+!")

        print("Your letter grade is A!")
